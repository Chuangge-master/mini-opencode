# 路线图

本文档列出了 mini-OpenCode 的开发计划和即将推出的功能。欢迎社区贡献！

## 🚀 计划功能

### 核心能力
- [ ] **Agent 运行时沙箱环境**：实现一个安全的沙箱环境，用于执行 Agent 生成的代码，以确保安全性和隔离性。
- [ ] **澄清机制**：使 Agent 能够在用户指令模棱两可或不完整时主动提出澄清问题。
- [ ] **长期记忆**：实现持久记忆能力，允许 Agent 跨不同会话回忆项目上下文、用户偏好和历史记录。

### UI/UX 改进
- [ ] **子 Agent 可视化**：添加 UI 组件，以在复杂任务委派期间可视化子 Agent 的层级结构和活动。
- [ ] **推理过程展示**：支持展示模型内部的“思考”或“推理”过程（思维链），以提高透明度。
- [ ] **TUI 性能优化**：优化终端用户界面 (TUI) 渲染引擎，以减少延迟并提高响应速度。
