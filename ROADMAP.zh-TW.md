# 路線圖

本文件列出了 mini-OpenCode 的開發計畫和即將推出的功能。歡迎社群貢獻！

## 🚀 計畫功能

### 核心能力
- [ ] **Agent 執行時沙箱環境**：實現一個安全的沙箱環境，用於執行 Agent 產生的程式碼，以確保安全性和隔離性。
- [ ] **澄清機制**：使 Agent 能夠在使用者指令模稜兩可或不完整時主動提出澄清問題。
- [ ] **長期記憶**：實現持久記憶能力，允許 Agent 跨不同工作階段回憶專案上下文、使用者偏好和歷史記錄。

### UI/UX 改進
- [ ] **子 Agent 視覺化**：新增 UI 元件，以在複雜任務委派期間視覺化子 Agent 的層級結構和活動。
- [ ] **推理過程展示**：支援展示模型內部的「思考」或「推理」過程（思維鏈），以提高透明度。
- [ ] **TUI 效能最佳化**：最佳化終端使用者介面 (TUI) 渲染引擎，以減少延遲並提高回應速度。
